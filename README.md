# Moony Clicker

**Moony Clicker** is a browser-based incremental clicker game inspired by *Cookie Clicker*.  
The project was created as part of the **Ramon Projects** program and is fully built using HTML, CSS, and JavaScript.

The goal of the game is to collect cheese, upgrade your production, and progress through increasingly difficult levels using a custom progression system.

---

## Features
- Click-based resource generation (cheese).
- CPS and CPC upgrade system with dynamic scaling.
- Leveling system with increasing difficulty.
- Animated UI elements and popups.
- Random space-themed quiz events.
